## 🎯 Функционал

### Загрузка изображений:
1. Админ выбирает пол (male/female)
2. Выбирает категорию (current/target)
3. Выбирает номер типа (1-4)
4. Загружает файл
5. Файл отправляется в Telegram → получается `file_id`
6. `file_id` сохраняется в БД

### Удаление изображений:
1. Кнопка "Удалить" под каждым изображением
2. Подтверждение через `confirm()`
3. AJAX-запрос на `/delete_body_image/<id>`
4. Удаление из БД
5. Удаление карточки со страницы без перезагрузки

### UI/UX:
- ✅ Цветовая кодировка секций (primary, success, info, warning)
- ✅ Responsive grid (Bootstrap 4 колонки)
- ✅ Превью изображений 250px высотой
- ✅ Fallback для отсутствующих изображений (SVG placeholder)
- ✅ Индикатор загрузки "⏳ Загрузка..."
- ✅ Сообщения успеха/ошибки через `alert()`
