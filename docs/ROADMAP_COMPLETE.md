## 🚀 ЗАПУСК ПРОЕКТА

### 1. Настройка `.env`:
```env
# Telegram
TELEGRAM_BOT_TOKEN=your_bot_token
ADMIN_CHAT_ID=your_admin_chat_id

# OpenRouter AI
OPENROUTER_API_KEY=sk-or-v1-xxxxx

# Канал для подписки
REQUIRED_CHANNEL_ID=@your_channel
REQUIRED_CHANNEL_URL=https://t.me/your_channel

# Админка
ADMIN_PASSWORD_HASH=your_hash
SECRET_KEY=your_secret_key
```

### 2. Инициализация БД:
```bash
cd d:\NICOLAS\1_PROJECTS\_IT_Projects\_Калькулятор_КБЖУ_Лид_Магнит
python scripts/init_default_settings.py
```

### 3. Запуск бота:
```bash
python main.py
```

### 4. Запуск админки:
```bash
python start_admin_panel.py
```
Админка: http://127.0.0.1:8080

### 5. Загрузка изображений фигур:
- Открыть http://127.0.0.1:8080/body_images
- Загрузить 16 фотографий (4×4 категории)
- Проверить отображение
